Popover stays where it should

A fix for a wandering popover in 1.5.0.

## Fixed

- **The popover no longer drifts off the icon.** Showing or hiding the port count changes how wide the menu bar item is, which nudges it along the menu bar. The open popover stayed where it was and ended up next to the icon rather than under it. It now follows the icon, and it does the same whenever the item moves for any other reason.

## Install

Download below, unzip, and drag WhatPort.app to Applications, or:

```
brew upgrade --cask whatport
```

The app is signed and notarised by Apple.
