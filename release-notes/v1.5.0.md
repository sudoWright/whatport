A quieter menu bar

The port count next to the menu bar icon is now optional.

## New

- **Hide the port count.** A toggle in Settings drops the "2/4" from the menu bar and leaves the icon on its own. On by default, so nothing changes unless you want it to. Thanks to @ScherisMarie for the nudge.

## Fixed

- The menu bar item now announces itself properly to VoiceOver, whether or not the count is showing.

## Install

Download below, unzip, and drag WhatPort.app to Applications, or:

```
brew upgrade --cask whatport
```

The app is signed and notarised by Apple.
