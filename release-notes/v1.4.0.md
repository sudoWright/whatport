One-click updates, and bigger text

WhatPort can now resize its text, and when a new version is out it tells you and updates on a single click, so you are not chasing GitHub for downloads.

## New

- **Adjustable font size.** A slider in Settings scales the text across every panel, and the popover widens to match so nothing gets cramped or cut off.
- **Built-in updates.** When a newer version is available, WhatPort shows a banner with an Update button. One click downloads it, checks it is properly signed and notarised by Apple, and swaps itself out. Prefer to check on your own? There is a "Check for Updates…" item in the menu. Homebrew users can carry on with `brew upgrade`.

## Install

Download below, unzip, and drag WhatPort.app to Applications, or:

```
brew upgrade --cask whatport
```

The app is signed and notarised by Apple.
