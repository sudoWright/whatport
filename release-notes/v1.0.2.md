Fix the app failing to launch

WhatPort 1.0.1 didn't launch. You'd open it and nothing happened. This release fixes that. If you're on 1.0.1, upgrade.

## Fixes

- **The app launches.** 1.0.1 looked for one of its bundled resources in the wrong place inside the app and quit on launch before it ever reached the menu bar. It built and ran fine on my own machine, which is exactly why it slipped through. It now finds its resources in the right place, and the release build tests for this so it can't ship broken the same way again.

## Install

Download below, unzip, and drag WhatPort.app to Applications, or:

```
brew upgrade --cask whatport
```

The app is signed and notarised by Apple.
